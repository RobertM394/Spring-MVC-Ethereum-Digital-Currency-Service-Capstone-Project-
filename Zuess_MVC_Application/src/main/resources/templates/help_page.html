What is Blockchain technology?


Technology
Blockchain

A series of immutable data stored on a record managed by a cluster of computers. These machines share and create a decentralized system of data management. The ledgers are stored on nodes bound together with strong cryptographic concepts for security and time-stamped to update versions of the ledger.

Blockchain principles to know:

-Decentralized

-Transparent

-Immutable

Smart Contracts

Smart Contracts are technology made possible by blockchain. Essentially a contract written in code that when fufilled places its information in the blockchain making it immutable. These contracts are coded with specific context that when deployed is non reversable. Because this technology works on the blockchain all the pre-mentioned features of blockchain are applied here. For this project you explored how a smart contract functions as the rules for governing otter coin.
